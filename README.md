# 祝光军作品集 - 使用说明

## 🎨 设计更新 (v3 - 2026.05.04)

参考深色极简风格重新设计，主要更新：

### 视觉升级
- **纯黑背景** (#000000) + 白色/浅灰文字
- 极简导航栏，Logo 使用渐变效果
- 入场淡入动画 (IntersectionObserver)

### 新增功能
1. **Banner 自定义** - 右上角点击"🖼️ 更换Banner图"上传个人封面
2. **内容全部可编辑** - 关于我、工作经历、设计日志、联系方式均可直接点击编辑
3. **作品上传管理** - 在作品集底部上传新作品，自动分类筛选
4. **简历下载** - 一键生成文本格式简历（含所有工作经历）
5. **localStorage 持久化** - 所有修改自动保存，刷新不丢失

### 交互设计
- 工作历程默认折叠，点击展开时间轴流程图
- 作品卡片悬停上浮效果，删除按钮悬停显示
- 分类筛选即时切换

## 🚀 使用指南

### 1. 替换 Banner 图片
- 打开页面后，点击右上角"🖼️ 更换Banner图"
- 选择本地图片文件
- 图片会自动保存到浏览器

### 2. 上传作品图片
- 滚动到"作品集"区域
- 点击"+ 上传新作品"
- 填写标题和分类，选择图片文件
- 点击"添加作品"

### 3. 编辑工作经历
- 点击"工作历程"区域展开
- 直接点击任意文字即可编辑
- 编辑完成后点击其他地方自动保存
- 支持新增/删除/重置

### 4. 下载简历
- 点击导航栏"📄 下载简历"按钮
- 自动生成包含所有信息的文本简历

## 🎯 个性化定制

### 修改联系方式
直接点击联系方式区域，编辑后自动保存

### 修改关于我内容
直接点击简介文字，编辑后自动保存

### 添加技能标签
编辑 HTML 中 `.skill-tags` 区域的 `<span>` 元素

## 📱 响应式适配

页面已适配移动端：
- 导航栏自动堆叠
- 作品网格自适应
- 时间轴优化显示
